# Начни здесь (для новичка)

Как у простых программ: **скачал → запустил**.

---

## Способ 1. Готовый файл (самый простой)

1. Открой страницу релизов:
   https://github.com/reclaw17/goclaw-lite-cachyos/releases

2. Скачай файл:
   `GoClaw-Lite-x86_64.AppImage`

3. В папке с файлом открой терминал и выполни:

```bash
chmod +x GoClaw-Lite-x86_64.AppImage
./GoClaw-Lite-x86_64.AppImage
```

Если система ругается на FUSE:

```bash
./GoClaw-Lite-x86_64.AppImage --appimage-extract-and-run
```

> Если в Releases файла ещё нет — смотри Способ 2.

---

## Способ 2. Собрать на своём CachyOS (одна команда)

1. Скачай репозиторий с GitHub:
   - кнопка **Code → Download ZIP**
   - или:

```bash
git clone https://github.com/reclaw17/goclaw-lite-cachyos.git
```

2. Зайди в папку и запусти **один** скрипт:

```bash
cd goclaw-lite-cachyos
bash всё-сразу.sh
```

Скрипт сам:
- поставит нужное для сборки
- скачает исходники GoClaw
- соберёт Lite
- сделает AppImage

В конце скажет, где лежит файл.

---

## Если что-то пошло не так

Просто скопируй текст ошибки из терминала и пришли.
