НАЧНИ ЗДЕСЬ (CachyOS / Arch)
==============================

Ссылка на проект:
https://github.com/reclaw17/goclaw-lite-cachyos


ВАРИАНТ А — самый простой (без git)
-----------------------------------
1) Открой ссылку выше в браузере
2) Нажми зелёную кнопку Code → Download ZIP
3) Распакуй архив
4) Открой терминал в этой папке
   (правый клик → Открыть терминал)
5) По очереди выполни 4 команды:

   bash scripts/install-deps-cachyos.sh
   bash scripts/clone-upstream.sh
   bash scripts/build-lite.sh
   bash scripts/package-appimage.sh

6) Готовый файл появится здесь:
   dist/GoClaw-Lite-x86_64.AppImage

7) Скопируй его на флешку и запускай:
   chmod +x GoClaw-Lite-x86_64.AppImage
   ./GoClaw-Lite-x86_64.AppImage


ВАРИАНТ Б — через git
---------------------
git clone https://github.com/reclaw17/goclaw-lite-cachyos.git
cd goclaw-lite-cachyos
bash scripts/install-deps-cachyos.sh
bash scripts/clone-upstream.sh
bash scripts/build-lite.sh
bash scripts/package-appimage.sh


ЕСЛИ ОШИБКА
-----------
Скопируй текст из терминала и пришли его.
Ничего не ломай и не удаляй — разберём по логу.


ВАЖНО
-----
- install-deps нужен только на компьютере, ГДЕ СОБИРАЕШЬ
- на других ПК pacman-зависимости не ставятся
- на флешку кладётся AppImage
